# Gravity Form Templates

A collection of ready-to-use Gravity Form templates in JSON format. Browse by category, download the templates you need, and import them directly into your WordPress site.

## How to Use

1. **Browse** the category folders below to find a template that fits your needs
2. **Download** the JSON file (click the file, then click the "Download raw file" button)
3. **Import** into Gravity Forms:
   - In WordPress, go to **Forms → Import/Export**
   - Select the **Import Forms** tab
   - Choose the downloaded JSON file
   - Click **Import**
4. **Customize** the imported form to match your branding and requirements

## Template Features

All templates include:

- Pre-configured field validation
- Mobile-responsive layout
- Logical field organization
- Customizable confirmation messages

Some templates also feature:

- **Conditional logic** — Fields that show/hide based on user selections
- **Survey fields** — Likert scales and star ratings for feedback collection
- **Product fields** — For payment processing (requires Gravity Forms payment add-on)
- **File uploads** — For document and image submissions

## Requirements

- WordPress 5.0+
- Gravity Forms 2.5+ (templates use markup version 2)
- Gravity Forms Survey Add-On (for templates with Likert scales)
- Payment add-on (for donation/registration forms with payments)

## Customization Tips

After importing a template:

1. **Update branding** — Change form titles and descriptions to match your organization
2. **Adjust options** — Modify dropdown choices, checkbox options, and preset values
3. **Configure notifications** — Set up email notifications for form submissions
4. **Connect payments** — Link product fields to your payment gateway
5. **Add integrations** — Connect to your CRM, email marketing, or other tools
